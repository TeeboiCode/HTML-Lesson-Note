# HTML Lesson Note

A single-page beginner-friendly HTML lesson note website built with plain HTML, CSS, and a small amount of vanilla JavaScript.

## Overview

This project contains a complete lesson note experience for HTML beginners, organized into 11 chapters covering:

- Course introduction and page structure
- Elements, attributes, and semantic HTML
- Meta tags, text formatting, lists, images, and tables
- Forms, media, exercises, and chapter assessments

The page includes a fixed sidebar chapter navigation, active section highlighting while scrolling, a reading progress bar, and a scroll-to-top button.

## Project Structure

```text
.
├── index.html
├── README.md
└── assets/
    └── style.css
```

- `index.html`: main lesson note content and tiny interaction scripts
- `assets/style.css`: full visual styling (layout, typography, code blocks, tables, cards, responsiveness)

## Run Locally

No build tools or dependencies are required.

1. Clone or download this repository.
2. Open `index.html` directly in your browser.

Optional (recommended for local development):

- Use VS Code with the Live Server extension for auto-refresh while editing.

## Key Features

- Clean, chapter-based learning structure
- Polished dark-theme UI with responsive layout
- Styled code examples for readability
- Exercise and assessment blocks in each lesson
- Scroll progress indicator and active navigation state

## Customize

You can easily adapt this project for your own lesson notes:

- Update lesson content in `index.html`
- Adjust colors and typography variables in `assets/style.css` (`:root` section)
- Add or remove chapter links in the sidebar navigation

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
